# Project Overview

Trale is a privacy-respecting body weight diary app built with Flutter. It allows users to track their weight loss or gain progress with a modern Material Design 3 interface. The app emphasizes privacy by not collecting any data and being copyleft libre software.

Key features:
- Customizable themes and dark mode
- Multiple units support
- Progress tracking with data modeling
- Import/export measurements
- No permissions required
- No advertising or trackers

Tech stack: Flutter/Dart, with local storage using Hive, charts with FlChart, and other Flutter packages.

The project is structured as a standard Flutter app in the `app/` directory, with Android and iOS platform code.
